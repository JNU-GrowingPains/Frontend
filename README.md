# 성장통 - 상품 분석 대시보드

상품별 데이터 분석을 위한 대시보드 웹 애플리케이션입니다.

## 실행 방법

의존성을 설치하려면 `npm install`을 실행하세요.

개발 서버를 시작하려면 `npm run dev`를 실행하세요.

## 문서

프로젝트에 대한 자세한 문서는 [`docs/`](./docs/) 폴더를 참고하세요:

- **[README.md](./docs/README.md)**: 프로젝트 개요 및 주요 기능

- **[FILE_STRUCTURE.md](./docs/FILE_STRUCTURE.md)**: 파일 및 폴더 구조 상세 설명
